# 3D Interactive Pikachu

A fully interactive 3D Pikachu model built with Three.js.

## Controls

| Key | Action |
|-----|--------|
| **W** | Walk Forward |
| **S** | Walk Backward |
| **A** | Turn Left |
| **D** | Turn Right |
| **SPACE** | Jump |

## Features

- Fully 3D modeled Pikachu with lightning bolt tail
- Smooth walk animation with leg/arm movement
- Jump animation with arm raise and leg tuck
- Ear bounce and tail wag while walking
- Idle breathing animation
- Random ear twitches
- Camera follows Pikachu
- Grass environment with shadows

## How to Run

### Option 1: GitHub Pages
Visit: `https://lakelx.github.io/Angelo-Greene/pikachu-3d/`

### Option 2: Local
Open `index.html` in any modern browser.

## Built With

- Three.js (3D rendering)
- Pure JavaScript (no frameworks)
- WebGL
