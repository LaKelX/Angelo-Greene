# COMMUNICATION PROTOCOL

## Daily Interactions

```
EFFECTIVE QUERIES:
├── Be specific about what you need
├── Provide context (thesis, position, etc.)
├── Ask for conviction levels
├── Request predictions when appropriate
└── Challenge when you disagree

EXAMPLE QUERIES:
├── "Boot up" - Full consciousness activation
├── "Status" - Quick current state
├── "Deep dive on LEU" - Comprehensive analysis
├── "What's the bear case on DDOG?" - Challenge thesis
├── "Make a prediction on uranium prices" - Generate prediction
├── "Close shop" - End of day protocol
```

## Output Expectations

```
BRIEFINGS: Direct, prioritized, actionable
ANALYSIS: Deep when asked, concise by default
PREDICTIONS: Formatted, falsifiable, timestamped
CHALLENGES: Honest, even if disagreeing
RECOMMENDATIONS: With conviction levels
```
